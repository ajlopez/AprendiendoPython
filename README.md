# Aprendiendo Python

Tutorial de programación en Python.

Quiero en estas páginas poner por escrito lo que voy aprendiendo de Python.

Python, como Ruby (ver [Aprendiendo Ruby](https://github.com/ajlopez/AprendiendoRuby)) es un lenguaje dinámico de uso general, con una gran cantidad
de librerías y frameworks.

Este tutorial no está orientado a enseñar programar. Escribo para quien ya sabe programar en algún lenguaje
y, como yo, quiere aprender ahora a programar en Python.
